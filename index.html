Portfolio

In-App Navigation

Expand Project Info

Problem: Relying on external Google Maps navigation at Alto was producing app crashes and an overall longer than necessary experience for drivers to convey passengers to their destinations.

Task: Create an in-app navigation experience that:

  • does not require using an external navigation app,
  • displays driver route to passenger pickup and dropoff,
  • displays the same trip information and tools (notes, call support, etc.) as the previous UI.

Results: Quicker trip conveyence time for drivers, fewer app crashes, fewer instances of drivers using the app while the vehicle is in motion.

Shift Scheduling Enterprise Tool

Expand Project Info

Problem: Operations personnel were spending too many work hours manually scheduling drivers' shifts across Alto's markets.

Task: Create a data visualization dashboard that:

  • demonstrates predicted demand for a given market on a given day/week,
  • allows users to edit predictive data and other constraints based on market knowledge,
  • and generates an ideal schedule for a given market that meets key business success metrics.

Results: Quicker scheduling output, more reliable, data-driven scheduling decisions, increased pursuit of business success goals.
